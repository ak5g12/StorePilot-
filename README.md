# StorePilot

## Tech Stack
- Frontend: React + Vite
- Backend: Node.js + Express
- Database: MongoDB

## Folder Structure
- `frontend/`: React Vite application
- `backend/`: Express server
- `docs/`: Project documentation

## Run Commands
### Frontend
```bash
cd frontend
npm install
npm run dev
```

### Backend
```bash
cd backend
npm install
npm start
```
